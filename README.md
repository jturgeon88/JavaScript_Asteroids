# JavaScript_Asteroids

[LIVE DEMO](https://jturgeon88.github.io/JavaScript_Asteroids "JavaScript Asteroids Demo")

Arcade classic built for the browser using JavaScript and HTML Canvas element.

## Controls/Keys
* 'a' key for left
* 's' key for down 
* 'd' key for right
* 'w' key for up
* space-bar for fire
